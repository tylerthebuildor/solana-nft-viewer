# Solana NFT Viewer

## Setup

```bash
yarn install
yarn start
```

# Build from Scratch

```bash
npx create-react-app solana-nft-viewer
cd solana-nft-viewer

yarn add \
  @solana/web3.js \
  @solana/wallet-adapter-base \
  @solana/wallet-adapter-wallets \
  @solana/wallet-adapter-react \
  @nfteyez/sol-rayz-react \
  @nfteyez/sol-rayz
```
